O projeto foi desenvolvido na linguagem C como parte da disciplina Introdução à Programação de Computadores.

O objetivo foi aplicar as estruturas de repetição for, while e do-while para simular a movimentação básica de peças do xadrez.

A Torre se move cinco casas para a direita utilizando a estrutura for, o Bispo se move cinco casas na diagonal utilizando while e a Rainha se move oito casas para a esquerda utilizando do-while.

Os valores de movimentação foram definidos diretamente no código, sem entrada do usuário, conforme solicitado no desafio.

O programa exibe no console a direção do movimento a cada casa percorrida, com código comentado e organizado para facilitar o entendimento.
